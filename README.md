# Kathirvel S Portfolio

Modern, responsive portfolio built with React functional components and hooks, using only CSS for styling and animations.

## Run locally

1. Install dependencies:
   `npm install`
2. Start development server:
   `npm start`
3. Build production bundle:
   `npm run build`

## Important

Place the uploaded resume PDF in `public/` with this exact name:

`Kathirvel S - 24CS109 RESUME.pdf`
